# 本地画像整理

## 目标

生成或更新：

```txt
data/profile.local.json
```

该文件只服务本地岗位筛选和页面配置，不应被提交到仓库、写入公开岗位报告或回显不必要的敏感详情。

## 流程

1. 若用户尚未提供画像，优先执行 `python scripts/ensure_profile.py` 通过终端收集信息。
2. 用户在对话中直接提供信息时，仅将明确给出的字段写入画像；未知硬性条件保持为空，不进行猜测。
3. 画像字段应覆盖：学历/学位/专业/毕业年份/是否应届/政治面貌、证书与资格、户籍与生源地、工作状态、目标地区与岗位类别、学习基础与考试日期。
4. 若用户接着要求岗位匹配，读取 `agents/skills/job-scan/SKILL.md` 并重新扫描；修改画像本身不会让页面自行联网更新。

## 隐私规则

- 姓名可以留空，除非用户明确要求保存。
- 不将原始画像复制到 `content/local/job/`、Markdown 报告、README 或 Git 提交中。
- 可在公开岗位输出中写岗位条件和判定结论，但避免出现用户毕业年份、单位、姓名等原始个人描述。
- 在提交或推送前确认 `.gitignore` 继续排除 `data/profile.local.json` 与 `data/job-tracking.local.json`。
